About

The goal of this site is to teach basic JavaScript concepts using simple games. It runs on an angular front end and uses express and node.js for the server.

All of the tutorials are pulled from markdown files hosted on Github. This makes creating a tutorial very easy. As long as the markdown is formatted correctly, once a link is placed in the side-nav the server will pull and style the appropriate tutorial! If you would like to contribute, please feel free to submit a pull request. Take a look at some of the markdown files for formatting examples.
